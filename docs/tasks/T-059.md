# T-059: E2E test harness

- **Status**: TODO
- **Milestone**: 19 — E2E Behavioral Test Suite: Foundation
- **Depends**: T-020
- **PRD Reference**: §14

## Description

Create `test/e2e/` directory with a shared test harness for behavioral E2E tests. Create `test/e2e/helpers.ts` that re-exports `useTestProject` and `writeSchema` from `src/testing/index.ts` and adds E2E-specific helpers: `runMigration(ctx)` (runs full pipeline and returns result), `queryDb(ctx, sql, params?)` (executes SQL against test DB), `assertTableExists(ctx, name)`, `assertColumnExists(ctx, table, column)`, `assertEnumValues(ctx, name, values)`, `getColumnInfo(ctx, table, column)` (returns type, nullable, default, generated). Create `test/e2e/setup.ts` for any shared setup/teardown. Add a smoke test in `test/e2e/smoke.test.ts` that creates a simple table and verifies it exists.

## Produces

- `test/e2e/helpers.ts`
- `test/e2e/setup.ts`
- `test/e2e/smoke.test.ts`
