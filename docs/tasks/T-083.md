- **Status**: TODO

## Description

Create `test/e2e/scaffold.test.ts`. Scaffold: (1) scaffoldInit creates directory structure (tables/, enums/, functions/, views/, roles/, mixins/, pre/, post/), (2) scaffoldPre creates timestamped pre-script, (3) scaffoldPost creates timestamped post-script, (4) scaffoldMixin creates mixin template. Generate: (5) generateFromDb produces YAML for existing table (columns, indexes, constraints match), (6) generateFromDb produces YAML for enum, (7) generateFromDb produces YAML for function, (8) generateFromDb with --seeds captures row data, (9) Generated YAML round-trips: generate -> re-parse -> migrate to fresh DB -> drift shows no differences.
